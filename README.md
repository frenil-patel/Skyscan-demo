# SkyScan — IFF Demo UI

Interactive front-end demo for SkyScan, a software-defined Friend-or-Foe identification system for tactical UAS built as part of the Innovation for Defense Applications course at USC.

## What This Is

A browser-based prototype showing how SkyScan would look and feel in the field. Includes a live tactical map, HMAC authentication flow simulation, and a coordinated movement pattern demo.

## How To Use

Open `index.html` in any browser. No install or login required.

## Disclaimer

This is a UI demo only. Built for academic purposes as part of the H4D Hacking for Defense program. Does not represent a fielded or classified system.
